###################
Descarga la carpeta (zip) del proyecto, copialo en el servidor local corriendo con php 5.6.40, descomprimir la carpeta y renombrarla con "cotrolasistencia_api"
###################

********************************************
1. Abre el archivo /application/config/database.php y configura la conexión a la base de datos local
2. Abre el archivo /application/config/config.php y configura la base url de la api (http://localhost/controlasistencia_api/ ó https://controlasistencia_api.test/ (esta segunda opción para servidores locales con laragon))
********************************************



*******************
Requerimientos de servidor
*******************

Se recomienda PHP versión 5.6 o posterior.

También debería funcionar en 5.3.7, pero le recomendamos encarecidamente que NO ejecute
versiones tan antiguas de PHP, debido a la seguridad y el rendimiento potenciales
problemas, así como características que faltan.
